## Recommended Assets

Place your images and media files here:

- `video-poster.jpg` - Thumbnail image for the video player (recommended: 1920x1080)
- `favicon.ico` - Website favicon
- Other images for content cards

## Recommended Image Sizes

| Image Type | Recommended Size |
|------------|------------------|
| Video Poster | 1920x1080 (16:9) |
| Favicon | 32x32 or 64x64 |
| Content Images | 800x600 |

## APK Downloads

The `Dad_Medicine_Reminder_v2.apk` is an Android application that helps remind Khun Poo to take medicine. Users can download it directly from the homepage by clicking the green download button.

### Installing the APK
1. Download the APK from the website
2. On Android, enable "Install from unknown sources" in Settings
3. Open the downloaded APK file to install
