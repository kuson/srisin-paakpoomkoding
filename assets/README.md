# Assets Folder

Place your images and media files here:

- `video-poster.jpg` - Thumbnail image for the video player (recommended: 1920x1080)
- `favicon.ico` - Website favicon
- Other images for content cards

## Recommended Image Sizes

| Image Type | Recommended Size |
|------------|------------------|
| Video Poster | 1920x1080 (16:9) |
| Favicon | 32x32 or 64x64 |
| Content Images | 800x600 |
